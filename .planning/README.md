# .planning/ — agent scratchpad (gitignored)

Working memory for AI coding agents (Claude Code et al.). Put plans, task
breakdowns, scratch notes, and throwaway analysis here.

Rules:
- Everything in this folder **except this README** is gitignored — it never
  lands in commits or PRs.
- Treat files here as ephemeral; do not rely on them surviving a fresh clone.
- Do **not** store secrets here.
- Promote anything durable (decisions, conventions) up into `AGENTS.md` or
  `docs/` so it is tracked.
